# ML-Lab4

## Overview
This project focuses on data quality assessment and preprocessing using the Superstore dataset.

## Tasks
- Identified data quality issues (missing values and duplicates)
- Handled missing values using mean and median
- Detected and removed outliers using IQR
- Applied normalization (Min-Max and Z-score)
- Checked correlation and applied PCA only if needed

## Dataset
Superstore Sales Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Notes
PCA was not applied because the correlation between numerical features was weak.
